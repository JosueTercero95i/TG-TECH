# TG TECH - iPhone POS System

High-performance, touch-optimized POS system for iPhone retail stores.

## Features
- ✨ Premium UI with glassmorphism and animations
- 📱 Fully responsive (Mobile & Desktop)
- 🖨️ Bluetooth Thermal Printing (ESC/POS)
- 📊 Sales History with local persistence
- 🏷️ Custom Business Name editing

## Tech Stack
- Vanilla HTML/CSS/JS
- Bluetooth Web API
- LocalStorage for persistence

## Deployment
This project is ready to be deployed on GitHub Pages.
